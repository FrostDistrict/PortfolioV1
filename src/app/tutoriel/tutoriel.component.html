Mini Tutoriel Angular

C'est quoi Angular?

Angular est un framework Open source très populaire maintenu par l'équipe Angular a Google. Il est basé sur le language TypeScript et offre une plate-forme de dévelopement d'application web puissant. Il simplifie grandement le developement avec plusieurs fonctionnalitées telles que l'injection de dépendance, la programmation déclarative et plusieurs autres.

L'architecture de Angular consiste de tableaux qui gerent les declarations, les importations, les fournisseurs et les composantes. Malgré le fait d'utiliser qu'une seule page, il peut y avoir beaucoup de composantes html qui peuvent être chargé independemment.

Commandes utiles :

Nouvelle Application :

ng new ApplicationName --routing --skipTests

Generer une composante :

ng g c ma-composante

Generer une classe :

ng g cl ma-classe
